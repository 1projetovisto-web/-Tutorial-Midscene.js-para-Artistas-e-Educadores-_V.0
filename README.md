# Midscene.js + p5.js: Tutorial para Artistas e Educadores

Este repositório é um guia prático para arte-educadores e artistas interessados em explorar a automação e o pensamento computacional através do **Midscene.js**.

## 🚀 O que é este projeto?
O Midscene.js é um operador de IA de código aberto. Diferente de assistentes que apenas escrevem texto, o Midscene **age** na interface. Para o artista, isso permite focar na intenção criativa ("deixe o visual mais agressivo") enquanto a IA manipula os parâmetros técnicos.

## 🛠️ Tecnologias
- **Midscene.js**: Automação multimodal.
- **p5.js**: Geometria generativa e visual.
- **Node.js**: Para rodar os scripts de automação.

## 📂 Estrutura do Repo
- `/src`: Código fonte do canvas p5.js.
- `/automation`: Scripts de automação Midscene.
- `index.html`: Página principal com o tutorial interativo.
